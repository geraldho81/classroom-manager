# The Classroom Manager

## Stack
- Next.js, React, TypeScript
- Supabase for auth + database (@supabase/ssr, @supabase/supabase-js)
- shadcn/ui components (Radix + class-variance-authority + clsx)
- Lucide React for icons

## Notes
- Use @supabase/ssr for server-side Supabase calls (not the browser client)
- UI components follow shadcn/ui patterns — don't introduce other component libraries
